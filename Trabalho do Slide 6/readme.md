# Sistemas Distribuídos

## Pré-requisitos

1. Python instalado
2. Baixar as bibliotecas necessárias com o comando no console ```pip install lxml zeep werkzeug```
3. Baixar também a biblioteca spyne em sua versão atualizada (para a atual versão do Python) direto do Github ```pip install git+https://github.com/arskom/spyne.git``` (é necessário ter o Git CLI instalado)
